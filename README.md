一款轻量的 Windows 桌面喝水提醒应用。每天 10:00 至 18:00，每个整点用水面上升动画提醒你喝水。

## 下载

前往 [Releases](https://github.com/cassielee6688/water-reminder/releases/latest) 下载最新版：

- `喝水提醒-安装版-1.0.0.exe`：推荐普通用户使用。
- `喝水提醒-便携版-1.0.0.exe`：无需安装，下载后直接运行。
- `SHA256SUMS.txt`：用于校验下载文件完整性。

> 当前版本未购买 Windows 代码签名证书。首次运行时，Windows 可能显示 SmartScreen 提示。请确认文件来自本仓库的 Releases 页面，再点击“更多信息”与“仍要运行”。

## 功能

- 每天 `10:00-18:00` 每个整点提醒一次，共 9 次。
- 提醒视频约 10 秒内由小窗丝滑放大至全屏。
- 点击水滴按钮记录“喝了”，今日喝水次数加一。
- 点击暂停按钮后，今天不再提醒，第二天自动恢复。
- 关闭主窗口后继续驻留系统托盘。
- 支持用户自行开启或关闭开机自启。
- 当天喝水次数和停止提醒状态可跨重启保留。

## 使用说明

### 安装版

1. 下载并运行 `喝水提醒-安装版-1.0.0.exe`。
2. 按安装向导完成安装。
3. 可在主界面选择是否开启“开机自启”。

### 便携版

1. 下载 `喝水提醒-便携版-1.0.0.exe`。
2. 将文件放到一个固定目录后运行。
3. 如果开启开机自启，请勿再移动或重命名该 exe，否则 Windows 启动项会失效。

## 本地开发

需要 Node.js 20 或更高版本。

```powershell
npm ci
npm start
```

测试和打包：

```powershell
npm test
npm run smoke
npm run dist
```

## 隐私

应用不需要账号，不连接云端，不上传喝水记录，也不调用任何外部 API。所有状态仅保存在本机。
