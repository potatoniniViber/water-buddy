# 喝水提醒 v1.0.0

首个公开版本。

## 功能

- 每天 10:00 至 18:00，每个整点提醒喝水。
- 水面上升视频在约 10 秒内由小窗放大至全屏。
- 支持“喝了”和“今日不再提醒”两个操作。
- 记录当天喝水次数，第二天自动重置。
- 支持系统托盘后台运行和可选开机自启。
- 所有数据仅保存在本机。

## 下载建议

- 普通用户请选择 `喝水提醒-安装版-1.0.0.exe`。
- 不想安装时请选择 `喝水提醒-便携版-1.0.0.exe`。
- 便携版开启开机自启后，请勿移动或重命名 exe。

## Windows 提示

当前版本未购买代码签名证书。Windows 首次运行时可能显示 SmartScreen 提示，请确认文件来自本项目的 GitHub Releases 页面后，再选择“更多信息”与“仍要运行”。
